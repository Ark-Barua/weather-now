\# Contributor Covenant Code of Conduct



\## Our Pledge

We pledge to make participation in this project a harassment-free experience for everyone, regardless of age, body size, disability, ethnicity, gender identity, sexual orientation, nationality, religion, or experience level.



We want this project to be a welcoming and inclusive environment for all contributors.



\## Our Standards

Examples of behavior that contributes to a positive environment:



\- Being respectful and considerate  

\- Using welcoming and inclusive language  

\- Giving constructive feedback  

\- Showing empathy toward others  



Examples of unacceptable behavior:



\- Harassment, insults, or personal attacks  

\- Publishing private information without permission  

\- Any conduct that would be considered inappropriate in a professional setting  



\## Our Responsibilities

Project maintainers must:



\- Clarify standards of acceptable behavior  

\- Take corrective action when inappropriate behavior occurs  

\- Remove or reject contributions that violate this Code of Conduct  



\## Scope

This Code of Conduct applies within:



\- Project repositories  

\- Issue trackers  

\- Pull requests  

\- Discussions on any project-related platform
